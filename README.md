# FACEENTRYATTEND-AI

AI-based Face Recognition Attendance System using Python, OpenCV, KNN, and Streamlit.

## Features
- Face detection using Haar Cascade
- Face recognition using KNN
- Automatic attendance marking
- Real-time webcam detection
- Attendance stored in CSV format

## Technologies Used
- Python
- OpenCV
- NumPy
- Streamlit
- Scikit-learn

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
